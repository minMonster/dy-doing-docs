# v-audio

## Description

音频播放组件

组件内部限制同一类型的组件只会有一个播放

## Attributes

| Attribute |   Description    |  Type   | Accepted values | Default |    Demo    |
| :-------: | :--------------: | :-----: | :-------------: | :-----: | :--------: |
|    id     |  audio 标签 id   | String  |        —        |   ''    | 'audioId'  |
|    src    |    audio src     | String  |        —        |   ''    | 'demo.mp3' |
|   desc    | 秒数自增，或自减 | Boolean |   true, false   |  true   |    true    |
|    drt    |    音频总时间    | Number  |        —        |    0    |    100     |
